# boardify

JSON based single page static board generator for links and short notes

## demo

https://mkkhedawat.github.io/boardify-demo/

## how to use

### install
```sh
$ npm install -g boardify
$ boardify --help
```

### run

```sh
$ boardify --init
$ cd myBoard
$ boardify --run

```

### build

```sh
$ boardify --build
```

## contribution

```bash
- clone forked boardifyRepository
- cd boardifyRepository
- make changes
- npm link
- test changes
- raise pull request

```
